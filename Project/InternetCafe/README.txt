#tkinterนี้ไม่ได้ทำแบบresponsive 
เพื่อLayoutที่สวยงานโปรดตั้งค่าที่
Settings > Display > แก้ไข > 1.Scale and layout แก้เป็น 100% 
> 2.resolution 1920x1800

ขั้นตอนการใช้งาน
1. Add Computer (ใส่ได้สูงสุด 45เครื่อง)
2. Adjust Rate (ปรับเรทราคาต่อชั่วโมง ?บาท/ชั่วโมง)
3. Search (ใส่เลขเครื่องที่ต้องการใช้งาน)
4. กดปุ่ม “เปิดใช้งาน”
5. ใส่จำนวนชั่วโมงที่ต้องการ
6 กดปุ่ม “ปิดใช้งาน” เพื่อยกเลิกหรือล้างเวลา
